  • Git Ekleme Güncelleme
  • Önce github hesabından "repositories" oluşturuyoruz.
  • Bash terminali ile projenin olduğu klasöre giriş yapıyoruz. cd /proje/dizin/yolu
  • Bir depo başlatıyoruz. git init
  • Tüm dosyaları içine alıyoruz. git add .
  • İlk commit yani yükleme işlemini yapıyoruz. git commit -m "İlk commit"
  • Oluşturduğumuz Github deposunu ekliyoruz git remote add origin https://github.com/kullanici_adi/my-project.git
  • Değişiklikleri githuba gönderiyoruz. git push -u origin master
  • Değişiklikleri 3 satırda tek seferde güncellemek için. 
                        git add .
                    git commit -m "Değişiklik açıklaması"
                    git push
